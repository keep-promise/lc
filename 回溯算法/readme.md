# 回溯算法模板
回溯法解决的问题都可以抽象为树形结构（N叉树），用树形结构来理解回溯就容易多了
```js
const dfs = backtracking(参数) => {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
